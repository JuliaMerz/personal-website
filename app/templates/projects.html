ECC Problem Generator

This was made after a computer science professor at UC Berkeley wanted someone to take an offline script that printed some ECC problems and put it online. I converted the script to use a web form instead of stdin for input and display errors in a more graceful manner. Then I stuck it into a Flask server and put it up here.
